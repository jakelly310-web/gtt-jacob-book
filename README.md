# GTT Canada — Book a meeting

Public, GTT-branded landing page for booking a 20-minute intro with Jacob Kelly
(Channel Sales Director, Canada). The "Book your 20 minutes" button links to
Jacob's Microsoft Bookings page; this page wraps it in GTT branding for the
Canada partner campaign CTAs and email signature.

Static site (single `index.html` + assets). No build step. Deploy on Netlify
with publish dir `.` (see `netlify.toml`).
