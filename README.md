# GTT — Book a meeting with Jacob Kelly

Public, GTT-branded landing page for booking a 20-minute intro with Jacob Kelly
(Channel Sales Director, GTT). The "Book your 20 minutes" button links to
Jacob's Microsoft Bookings page; this page wraps it in GTT branding. Used as the
CTA target across all of Jacob's outreach (partners and distributed-enterprise
customers) and motions in Pennsylvania, Upstate New York, and Canada, plus his
HiHello email signature.

Static site (single `index.html` + assets). No build step. Deploy on Netlify
with publish dir `.` (see `netlify.toml`).
